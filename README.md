# sugimori-arg
# 杉守村 ARGゲーム

## 概要

本作品は、架空の山村「杉守村」を舞台にした、Web探索型のARG風ゲームです。

プレイヤーは、検索エンジン風のページ「Moorii」を使って情報を集めながら、突然連絡が取れなくなった友人「杉谷遥」の行方を追っていきます。

検索結果、掲示板、チャットログ、公式サイト風のページなどを読み進めることで、杉守村に隠された違和感が少しずつ見えてきます。

作品内に登場する人物・団体・地名・事件はすべてフィクションです。

## 遊び方

1. `index.html` を開いてください。
2. Moorii の検索欄に、気になる言葉を入力してください。
3. 検索結果やページ内リンクをたどりながら、情報を集めてください。
4. ときには、ページ内の小さな違和感や言葉が次の手がかりになることがあります。
5. あなたの調査の進め方によって、物語は変化します。

## ヒント

この作品では、すべての情報が最初から見えるわけではありません。

同じ場所や人物について調べる場合でも、入力する言葉によって表示される情報が変わることがあります。
ページ内に出てきた言葉、気になった表現、誰かの名前などを、もう一度検索してみてください。

また、一部のページでは、普通に読むだけではなく、ページ内の検索欄やボタンを使うことで新しい情報が見つかる場合があります。

## 制作テーマ

本作品では、「検索する」「リンクをたどる」「チャットを読む」といった、普段のインターネット利用に近い行動をゲーム体験として取り入れました。

普通の観光情報や地域情報の中に、少しずつ不自然な情報が混ざっていくことで、プレイヤー自身が調査しているような感覚を出すことを目指しています。

また、ホラー表現を直接的に見せるのではなく、Webページの形式や削除された情報、会話ログの沈黙などを通して、不安感が積み重なるように構成しました。

## AIの使用について

本作品の制作では、ChatGPTを使用して以下の作業を行いました。

* 物語設定やゲーム構成の整理
* 架空のWebページやチャットログの文章作成
* HTML / CSS / JavaScript の作成と修正
* 検索システムやページ遷移の調整
* 作品内で使用する画像・ビジュアル案の作成補助

AIが生成した内容をそのまま使用するのではなく、作品の雰囲気や展開に合わせて、文章・UI・導線を何度も修正しながら制作しました。

## 注意事項

この作品は完全なフィクションです。
実在の人物・団体・地名・事件・信仰・医療機関などとは一切関係ありません。

失踪・事件を扱う表現が含まれます。苦手な方は注意してください。
